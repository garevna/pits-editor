# dgtek-portal-polygons-editor

### Install
```
yarn add dgtek-portal-polygons-editor
```

### Import

```
import 'dgtek-portal-polygons-editor'
import 'dgtek-portal-polygons-editor/dist/dgtek-polygons.css'
```

### Usage

```
<Polygons
  :host="host"
  :apiKey="apiKey"
  :credentials="credentials"
/>
```
